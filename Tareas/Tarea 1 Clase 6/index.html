Encabezados (h1 - h6)

Las etiquetas <h1> to <h6> se utilizan para definir encabezados HTML.
<h1> Define el título más importante.
<h6> Define el encabezado menos importante.
Nota: Utilice solo uno <h1> por página; este debe representar el título/asunto principal de toda la página. Además, no se salte los niveles de encabezado: comience con <h1>, luego use <h2>, y así sucesivamente.

Ejemplo de uso

<h1> encabezado </h1>

Parrafos (p)

La etiqueta <p> define un párrafo. Los navegadores agregan automáticamente una sola línea en blanco antes y después de cada elemento <p>.

Ejemplo de uso

<p> parrafo </p>

Time

La etiqueta <Time> define una hora específica (o fecha y hora). El atributo de fecha y hora de este elemento se utiliza para traducir la hora a un formato legible por máquina para que los navegadores puedan ofrecer agregar recordatorios de fechas a través del calendario del usuario y los motores de búsqueda puedan producir resultados de búsqueda más inteligentes.

Ejemplo de uso

<time> <p> Open from <time> 10:00 </time> to <time> 21:00 </time> every weekday.</p> </time>

Video

La etiqueta <video> se utiliza para incrustar contenido de video en un documento, como un clip de película u otras secuencias de video. La etiqueta <video> contiene una o más etiquetas <source> con diferentes fuentes de video. El navegador elegirá la primera fuente que admita. El texto entre las etiquetas <video> y </video> solo se mostrará en navegadores que no admitan el elemento <video>. Hay tres formatos de vídeo compatibles con HTML: MP4, WebM y OGG.

Ejemplo de uso

<video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> <source src="movie.ogg" type="video/ogg"> Your browser does not support the video tag. </video>

Object

La etiqueta <object> define un contenedor para un recurso externo. El recurso externo puede ser una página web, una imagen, un reproductor multimedia o una aplicación complementaria.

Ejemplo de uso

<object data="pic_trulli.jpg" width="300" height="200"></object>

Menu

La etiqueta <menu> define una lista desordenada de contenido. Utilice la etiqueta <menu> junto con la etiqueta <li> para crear elementos de menú.

Ejemplo de uso

<menu> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </menu>

Hgroup

La etiqueta <hgroup> se utiliza para rodear un encabezado y uno o más elementos <p>. El encabezado dentro del elemento <hgroup> puede ser cualquiera de los encabezados <h1> a <h6>.

Ejemplo de uso

<hgroup> <h2>Norway</h2> <p>The land with the midnight sun.</p> </hgroup>

Dialog

La etiqueta <dialog> define un cuadro de diálogo o una subventana. El elemento <dialog> facilita la creación de cuadros de diálogo emergentes y modales en una página web.

Ejemplo de uso

<dialog open>This is an open dialog window</dialog>

Audio

La etiqueta <audio> se utiliza para incrustar contenido de sonido en un documento, como música u otras transmisiones de audio. La etiqueta <audio> contiene una o más etiquetas <source> con diferentes fuentes de audio. El navegador elegirá la primera fuente que admita. El texto entre las etiquetas <audio> y </audio> solo se mostrará en navegadores que no admitan el elemento <audio>. Hay tres formatos de audio compatibles en HTML: MP3, WAV y OGG.

Ejemplo de uso

<audio controls> <source src="horse.ogg" type="audio/ogg"> <source src="horse.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio>

Adress

La etiqueta <address> define la información de contacto del autor/propietario de un documento o artículo. La información de contacto puede ser una dirección de correo electrónico, URL, dirección física, número de teléfono, identificador de redes sociales, etc. El texto del elemento <address> normalmente se muestra en cursiva y los navegadores siempre agregarán un salto de línea antes y después del elemento <address>.

Ejemplo de uso

<address> Written by <a href="mailto:webmaster@example.com">Jon Doe</a>.<br> Visit us at:<br> Example.com<br> Box 564, Disneyland<br> USA </address>
